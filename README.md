# 🤖 Telegram Bot + Free Neural Network

Этот простой Telegram-бот написан на Python и использует библиотеку, предоставляющую бесплатный доступ к ChatGPT для обработки запросов пользователей. Бот может отвечать на вопросы, генерировать текст, выполнять другие задачи с помощью языковой модели — без затрат на платные API.

## 🚀 Возможности

- Принимает текстовые сообщения от пользователей
- Отправляет запрос к нейросети 
- Возвращает сгенерированный ответ в Telegram пользователю
- Легко расширяется под другие задачи

## 🛠 Используемые технологии

- Python 3.12
- Aiogram
- g4f

## 📦 Установка

1. Клонируйте репозиторий:
   git clone https://github.com/SH1ROdev/Free-GPT-TgBot
   cd Free-GPT-TgBot

2. pip install -r requirements.txt
В main вставьте токен бота или создайте .env чуть изменив код

3. Запустите бота:
python main.py

⚙️ Настройка нейросети
По умолчанию бот использует g4o-mini.
